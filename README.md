# AlphaDesk 📊

Personal trade analytics & portfolio tracking platform. Built as a single HTML file, powered by Supabase.

## Features

### Analytics & Tracking
- Dashboard — cumulative PnL, win rate, market distribution charts
- PnL Calendar — daily profit/loss on a monthly calendar view
- Detailed Analysis — risk/reward, profit factor, best/worst trades
- Leaderboard — compare performance across users

### Market Data
- Portfolio — NASDAQ, NYSE, BIST & crypto live price tracking
- Search any symbol → live price loads → P/L preview → add
- Commodities — gold, silver, oil, gas, copper, platinum & more
- Live Ticker — BTC, ETH, EUR/USD, XAU/USD, S&P500, BIST100 and more

### Communication & News
- Forex Calendar — economic events with impact levels & currency filters
- Market News — Investing.com, ForexLive, MarketWatch, CoinDesk
- Group Chat — real-time messaging with image sharing & delete
- Direct Messages — private 1-on-1 messaging between users
- Notes — personal trade notes & strategy journal

### System
- Multi-user — register, login, change nickname/password
- Admin Panel — user management, message moderation, system stats
- Cloud sync — access from any device via Supabase
- TR / EN language toggle
- Dark / Light mode

## Stack

| Layer | Technology |
|---|---|
| Frontend | Vanilla HTML / CSS / JavaScript |
| Database | Supabase (PostgreSQL) |
| Price Data | Yahoo Finance API |
| Charts | Chart.js |
| Hosting | GitHub Pages |

## Setup

Deploy `index.html` to GitHub Pages. Supabase connection is pre-configured — no additional setup required.
